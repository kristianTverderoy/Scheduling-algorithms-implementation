# Scheduling-algorithms-implementation
Assignment 2 for OS subject.

## How to run:
This project runs through tests with printed output in the output terminal

Test classes are located in "/app/src/test/java/org/ntnu/*" with each test class representing the three test cases

Using Visual Studio Code:
Either right click the test class you want to run and select run tests, or enter each test class and run specific methods with the run button to the left of each test method.
