# Scheduling-algorithms-implementation
Assignment 2 for OS subject.

## How to run:
This project runs through tests with printed output in the DEBUG CONSOLE

Test classes are located in "/app/src/test/java/org/ntnu/*" with each test class representing the three test cases.

To run the tests:
Either right click the test class you want to run and select run tests, or enter each test class and run specific methods with the run button to the left of each test method. The printed results should be in the DEBUG CONSOLE tab in the terminal. (note: running all tests at once cuts some the result of the first algorithm.)
